[h1]Description[/h1]
This mod enhances block renaming in Space Engineers by adding advanced controls to thrusters and other blocks, allowing you to rename multiple blocks at once with ease. Inspired by two excellent simple renaming mods:
[list]
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=2077166496]Easy Block Renaming[/url] by Meridius_IX / Lucas ([url=https://steamcommunity.com/id/meridius_ix]profile[/url]) – the original author has retired from modding.
[*][url=https://steamcommunity.com/sharedfiles/filedetails/?id=2149516482]Advanced Block Renaming (Regex Renamer)[/url] by Marcy ([url=https://steamcommunity.com/id/stopsearchingrandomurls]profile[/url]) – a great regex-based tool that served as additional inspiration.
[/list]
My version builds on these ideas with new features like directional thruster renaming, group-based numbering, and a centralized numbering logic tailored for organizing complex builds.

[h1]How to Use This Mod[/h1]
[list]
[*]Subscribe and add it to your world's mod list.
[*]Select one or more blocks in the terminal.
[*]Click the [b]Show/Hide Rename Controls[/b] button at the bottom (controls are hidden by default to keep the panel clean). Set your template and apply!
[/list]

[h1]Features[/h1]
[list]
[*][b]Directional Thruster Renaming[/b] – Renames based on thrust direction (F, B, U, D, L, R) using "{0}" template.
[*][b]Sequential Numbering[/b] – Advanced numbering with adjustable digit format (e.g., "001").
[*][b]Auto-Continue & Grouping[/b] – Smart counters that remember where you left off per grid or per block type.
[*][b]Flexible Separators[/b] – Full control over the character between name and number, including completely empty separators.
[/list]

[h1]Important: Terminal Input Limitations[/h1]
[b]Note for non-US Keyboard users (e.g., Croatian, German, etc.):[/b]
The Space Engineers terminal [b]does not allow[/b] the input of characters that require the [b]AltGr[/b] key combination.
[list]
[*] Characters like [b] \ | [ ] { } < > [/b] cannot be typed directly into the mod's textboxes if your layout requires AltGr (e.g., [b]AltGr + Q[/b] for backslash).
[*] [b]Workaround:[/b] If you need these characters for Regex patterns, type them in Notepad and use [b]Ctrl+V[/b] to paste them into the game's textbox.
[/list]

[h1]Regex Renaming Guide[/h1]
Regex (Regular Expressions) is a powerful way to bulk-replace parts of names using patterns. The mod uses [b]C# .NET syntax[/b].
[list]
[*] [b]Wildcards:[/b] [b].[/b] (any single char), [b].[/b] (everything)
[*] [b]Groups:[/b] Use b[/b] to capture parts of the name, and [b]$1, $2, $3[/b] to replace them.
[*] [b]No quotes needed:[/b] Just enter the pattern directly into the textboxes.
[/list]
[b]Common Pattern Examples:[/b]
[table]
[tr]
[th]Goal[/th]
[th]Find (Pattern)[/th]
[th]Replace With[/th]
[th]Example Result[/th]
[/tr]
[tr]
[td]Clean suffix[/td]
[td][code]Thruster.[/code][/td]
[td][code]Thruster[/code][/td]
[td]Thruster Large Atmo → Thruster[/td]
[/tr]
[tr]
[td]Remove numbers[/td]
[td][code]\d+[/code][/td]
[td](leave empty)[/td]
[td]Gyroscope 123 → Gyroscope[/td]
[/tr]
[tr]
[td]Add Prefix[/td]
[td]code[/code][/td]
[td][code]Main $1[/code][/td]
[td]Battery → Main Battery[/td]
[/tr]
[tr]
[td]Swap words[/td]
[td]code (\w+)[/code][/td]
[td][code]$2 $1[/code][/td]
[td]Hydrogen Tank → Tank Hydrogen[/td]
[/tr]
[tr]
[td]Remove after dash[/td]
[td][code] -.[/code][/td]
[td](leave empty)[/td]
[td]Connector - Left → Connector[/td]
[/tr]
[/table]
